Conseil: Ouvrez l'onglet "Network" dans les outils de développement avant de cliquer sur les boutons.

Pour observer les appels réseau:

  1. Ouvrez les outils de développement (F12)
  2. Allez dans l'onglet "Network"
  3. Filtrez par "Fetch/XHR"
  4. Exécutez un swap
  5. Observez les requêtes vers api.thorswap.net

Vous pourrez voir les détails de la requête (Headers, Payload, Response) en cliquant sur chaque appel.
